# 🧠 PsyCare (Em Desenvolvimento)

O **PsyCare** é uma plataforma Full-Stack projetada para conectar psicólogos e pacientes, facilitando o agendamento de consultas, o acompanhamento terapêutico e a gestão de prontuários de forma segura e intuitiva.

---

## 🚧 Status do Projeto
O projeto está atualmente na fase inicial de desenvolvimento (**MVP - Minimum Viable Product**). 
As funcionalidades de autenticação e estrutura base do banco de dados estão sendo priorizadas.

---

## 🎯 Objetivos do Projeto
* **Gestão de Consultas:** Agendamento inteligente e lembretes automáticos.
* **Segurança de Dados:** Implementação de criptografia para proteger informações sensíveis de pacientes.
* **Interface Humanizada:** Design focado em acessibilidade e conforto visual, essencial para o contexto de saúde mental.

---

## 🛠️ Tecnologias Planejadas

* **Frontend:** [Next.js](https://nextjs.org/) (App Router) + [Tailwind CSS](https://tailwindcss.com/)
* **Backend:** [Node.js](https://nodejs.org/) com [Next.js Server Actions](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations)
* **Banco de Dados:** [PostgreSQL](https://www.postgresql.org/) (via [Supabase](https://supabase.com/))
* **ORM:** [Prisma](https://www.prisma.io/)
* **Autenticação:** [Clerk](https://clerk.com/) ou [NextAuth.js](https://next-auth.js.org/)
* **Estilização de Componentes:** [Shadcn/UI](https://ui.shadcn.com/)

## 📦 Como testar o que já foi feito (Local)

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Dot010/psycare.git](https://github.com/Dot010/psycare.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o ambiente:**
    ```bash
    npm run dev
    ```
